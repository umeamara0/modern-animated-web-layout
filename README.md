Modern Animated Layout with Theme Toggle
A professional, visually appealing web interface that showcases creative UI/UX design, integration of external animation libraries, and dynamic theme switching logic.

 Key Features
 Dark & Light Mode**: A fully functional theme switcher implemented using **JavaScript DOM manipulation**.
 Smooth Animations**: Integrated **Animate.style** library for high-quality entrance effects and motion graphics.

 Tech Stack

* **HTML5**: Used for semantic structure and accessible web content.
* **CSS3**: Custom styling, Flexbox/Grid for layout, and CSS variables for theme management.
* **Animate.style**: Used for professional-grade, ready-to-use CSS animations.
* **JavaScript (ES6)**: Logic building for the theme toggle and interactive elements.

 📂 Project Structure

- `index.html` - The main structure of the page.
- `style.css` - Custom styles and theme variables.
- `js/index.js` - JavaScript logic for Dark/Light mode.
